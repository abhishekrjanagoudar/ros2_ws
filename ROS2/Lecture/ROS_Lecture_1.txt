Why ROS ?

• Industry Standard & Employability

− most top-tier robotics companies use ROS2
− „must have“ skill on resume for internships and full-time roles

• Hardware Independence

− write code that can be easily ported between robots

• Massive Community & Ecosystem

− don‘t reinvent
− access pre-built packages for mapping, navigation, planning, computer vision

• Advanced Middleware

− ROS2 uses Data Distribution Service (DDS)
− teaches how modern distributed systems handle real-time communication and

− integrates with simulators like Gazebo and Isaac Sim
− test complex AI and control algorithms in a virtual world before risking expensive hardware

The Python Philosophy :: Info for C++ Developers

• Design Philosophy

− Python prioritizes readability and development speed (productivity) over fine-grained memory control.
Think of it as "executable pseudocode."

• Memory Management

− Forget new, delete, and smart pointers. Python uses automatic garbage collection and reference counting.

• Typing System

− Python is strongly but dynamically typed. Variables are names pointing to objects, not fixed memory locations.
No more int x = 5; , just x = 5.

• Execution Model

− It is an interpreted/bytecode-compiled language.
There is no separate "compile" step like g++; the interpreter handles it at runtime.

• Syntax Shift

− Braces vs. Indentation: Whitespace defines scope (blocks). No more ; or {}.
− Everything is an Object: Even integers and functions are objects on the heap.

• The "Standard Library" Advantage

− Tasks requiring external libraries in C++ (like JSON parsing or HTTP requests) are often built-in or one-liners in Python.
In C++, you tell the computer how to do it.
In Python, you tell the computer what you want done.
